### File Type Display in Drop Zone

The supported formats line in the drop zone must be updated from:
```
Supports: .txt, .md, .pdf, .docx, .html
```
to:
```
Supports: .txt, .md, .pdf, .docx, .html, .jpg, .png, .xlsx
```

This change applies in three locations within `FileUpload.tsx`:
1. `ACCEPTED_TYPES` array -- add `.jpg`, `.jpeg`, `.png`, `.xlsx`
2. `<input accept="...">` attribute -- add `.jpg,.jpeg,.png,.xlsx`
3. Visible text in the drop zone `<p>` element and error message template string
