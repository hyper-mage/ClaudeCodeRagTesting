However, React still mounts BOTH trees in the DOM regardless of `display:none`. To avoid duplicate dnd-kit ids, conditionally mount the drawer tree only when `isDrawerOpen`: change `<MobileDrawer isOpen={isDrawerOpen} ...>` so its children only render when `isDrawerOpen` is true — wrap the children in `{isDrawerOpen && (<>...</>)}`. (The drawer panel itself stays mounted for transition smoothness; only the FolderTree subtree gates on `isDrawerOpen`.)
