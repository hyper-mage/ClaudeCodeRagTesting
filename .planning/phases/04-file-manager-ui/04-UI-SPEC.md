## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, tree indent per level, inline padding (gap-1, p-1) |
| sm | 8px | Tree item padding, compact element spacing (p-2, gap-2) |
| md | 12px | Sidebar section padding, list row padding (px-3, p-3) |
| lg | 16px | Content panel padding, form spacing (p-4, space-y-4) |
| xl | 24px | Section padding, upload zone inner padding (p-6) |
| 2xl | 48px | Upload zone large padding (p-12) |

Exceptions:
- Tree indent increment: 16px per depth level (ml-4 per level, starting at depth 0)
- Touch target minimum: 32px height for tree items (py-1.5 + content) to support drag handles
- Context menu items: 32px minimum height for click targets

## Typography

| Role | Size | Weight | Line Height | Tailwind Class |
|------|------|--------|-------------|----------------|
| Body | 14px | 400 (normal) | 1.5 | text-sm |
| Label | 12px | 400 (normal) | 1.5 | text-xs |
| Section heading | 14px | 500 (medium) | 1.5 | text-sm font-medium |
| Page heading | 24px | 700 (bold) | 1.2 | text-2xl font-bold |

Notes:
- Filenames in tree and list use Body (14px, 400)
- Status badges, chunk counts, metadata, and breadcrumb segments use Label (12px, 400)
- Sidebar section headers ("Board Games", "My Documents") use Section heading (14px, 500)
- No Display size needed for this phase

## Color

| Role | Value | Tailwind | Usage |
|------|-------|----------|-------|
| Dominant (60%) | #030712 | bg-gray-950 | Page background, content panel background |
| Secondary (30%) | #111827 | bg-gray-900 | Tree sidebar background, input backgrounds, card surfaces |
| Accent (10%) | #2563EB | bg-blue-600 | See reserved list below |
| Destructive | #DC2626 | bg-red-600 / text-red-400 | Delete confirmations, error messages, failed status badges |

Accent (blue-600) reserved for:
- Upload button (primary CTA)
- Drop target highlight border when dragging (border-blue-500, bg-blue-500/10)
- Focus ring on inputs (focus:border-blue-500)
- "processing" status badge background
- Breadcrumb active/clickable segment hover

Additional semantic colors already established in codebase:
- **Success:** bg-green-600 / text-green-100 -- "completed" status badge
- **Warning:** text-amber-400 -- duplicate file info message
- **Muted:** text-gray-500 -- disabled text, placeholder text, read-only indicators
- **Read-only indicator:** text-gray-500 + Shield icon at 12px -- Board Games tree items

### State Colors (interactions)

| State | Style | Usage |
|-------|-------|-------|
| Hover (tree item) | bg-gray-800 | Tree row hover, file list row hover |
| Selected (tree item) | bg-gray-800 text-white | Currently selected folder in tree |
| Drag ghost | opacity-50 | Semi-transparent clone of dragged element |
| Valid drop target | border-2 border-blue-500 bg-blue-500/10 | Folder accepting drop |
| Invalid drop target | cursor-not-allowed opacity-50 | Board Games folders during drag |
| Bulk selection active | bg-gray-800/50 | Selected rows with checkbox checked |
| Context menu | bg-gray-900 border border-gray-700 shadow-lg | Floating menu overlay |

## Interaction Contracts

### Tree Sidebar (FolderTree)

| Interaction | Behavior |
|-------------|----------|
| Click folder | Select folder, show contents in right panel, update breadcrumb |
| Click expand chevron | Toggle folder expanded/collapsed without selecting |
| Double-click folder name | Enter inline rename mode (private folders only) |
| Right-click folder | Show context menu at cursor position |
| Drag folder | Initiate drag with semi-transparent ghost |
| Drop on folder | Move dragged item into target folder (private only) |
| Drop external file on folder | Upload file to that folder |
| Keyboard: ArrowDown/Up | Move focus between visible tree items |
| Keyboard: ArrowRight | Expand focused folder |
| Keyboard: ArrowLeft | Collapse focused folder (or move to parent) |
| Keyboard: Enter | Select focused folder |
| Keyboard: F2 | Enter inline rename mode |
| Keyboard: Delete | Open delete confirmation |

### File List (FileListView)

| Interaction | Behavior |
|-------------|----------|
| Click row | Select single file (deselect others) |
| Click checkbox | Toggle file selection without deselecting others |
| Shift+click | Range select from last clicked to current |
| Right-click row | Show context menu |
| Double-click filename | Enter inline rename mode |
| Drag row | Initiate file move drag |
| Drop external file | Upload to current folder |

### Context Menu

| Interaction | Behavior |
|-------------|----------|
| Appears | At cursor position on right-click, max-width 200px |
| Click outside | Dismiss menu |
| Escape | Dismiss menu |
| Menu items (file) | Rename, Delete, Move to... |
| Menu items (folder) | New subfolder, Rename, Delete, Move to... |
| Menu items (Board Games) | None (no context menu on read-only items) |

### Inline Rename

| Interaction | Behavior |
|-------------|----------|
| Activation | Name text replaced by input, current name pre-filled and fully selected |
| Enter | Confirm rename, call API, revert to text display |
| Escape | Cancel rename, revert to original name |
| Click outside | Confirm rename (same as Enter) |
| Empty input + Enter | Cancel rename (no empty names allowed) |

### Breadcrumb

| Interaction | Behavior |
|-------------|----------|
| Click segment | Navigate to that folder, update tree selection |
| Last segment | Not clickable (current location, font-medium text-gray-200) |
| Other segments | Clickable, text-gray-400 hover:text-gray-200 |

### Confirm Dialog

| Interaction | Behavior |
|-------------|----------|
| Appearance | Centered modal with backdrop (bg-black/50) |
| Confirm button | "Delete" in red (bg-red-600 hover:bg-red-700) |
| Cancel button | "Cancel" in gray (bg-gray-700 hover:bg-gray-600) |
| Escape | Cancel and close |
| Click backdrop | Cancel and close |

### Bulk Action Bar

| Interaction | Behavior |
|-------------|----------|
| Appears | When 1+ items selected, fixed bar above file list |
| Content | "{N} selected" + "Delete" button + "Move to..." button |
| Escape | Deselect all, hide bar |
