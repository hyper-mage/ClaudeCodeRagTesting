    5. **`get_or_create_game_folder(db, game_name: str) -> str`** function:
       - Query folders table for existing folder with `parent_id = BOARD_GAMES_FOLDER_ID` and `name = game_name`
       - If found, return its id
       - If not found:
         - Generate a deterministic UUID using `uuid.uuid5(uuid.NAMESPACE_DNS, f"boardgame.{sanitize_ltree_label(game_name)}")`
         - Get the parent folder's ltree path: query folders table for BOARD_GAMES_FOLDER_ID, get its `path` value
         - Compute child path: `f"{parent_path}.{sanitize_ltree_label(game_name)}"`
         - INSERT into folders: `{"id": folder_id, "name": game_name, "parent_id": BOARD_GAMES_FOLDER_ID, "user_id": SYSTEM_USER_ID, "path": child_path, "visibility": "public"}`
         - Return the folder_id

    6. **`seed_game(db, filename: str, game_name: str) -> bool`** function:
       - Read file bytes from `os.path.join(DEFAULT_KB_DIR, filename)`
       - Compute content_hash via `hash_content(file_bytes)`
       - Check `check_duplicate(SYSTEM_USER_ID, content_hash)` -- if exists, print skip message and return False
       - Call `get_or_create_game_folder(db, game_name)` to get folder_id
       - Generate doc_id via `str(uuid.uuid4())`
       - Upload to Supabase Storage: `db.storage.from_("documents").upload(path=f"{SYSTEM_USER_ID}/{doc_id}/{filename}", file=file_bytes, file_options={"content-type": "text/markdown"})`
       - INSERT document record with: `id=doc_id, user_id=SYSTEM_USER_ID, filename=filename, storage_path=storage_path, file_size=len(file_bytes), mime_type="text/markdown", status="pending", content_hash=content_hash, folder_id=folder_id, visibility="public"`
       - Call `process_document(doc_id, SYSTEM_USER_ID)`
       - Print success message and return True

       - Print "Seeding default knowledge base..."
       - Iterate over GAMES dict, call `seed_game()` for each
