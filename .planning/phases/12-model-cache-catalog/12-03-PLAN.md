| Threat ID | Category | Component | Disposition | Mitigation Plan |

| T-12-V5-03 | Tampering / Input Validation | `free_only` query param | mitigate | Typed as `bool` in the route signature → FastAPI coerces/rejects non-bool input; no raw string flows into a query. Covered by test_free_only_filter. |
| T-12-V5-04 | Denial of Service | refresh path serving upstream data through the route | mitigate | build_model_response/refresh_if_stale (plan 12-01) parse defensively (D-11) and serve trimmed fields (~65 KB, not the full 535 KB raw — Pitfall 3); fetch bounded by httpx 10s timeout. | T-12-V4-03 | Information Disclosure | GET /api/models exposure | accept | Endpoint auth-gated (Depends get_user_id, A4) and serves only the non-secret public catalog — no per-user or secret data. Frontend reads here only, never OpenRouter directly (Success Criterion #1). |
